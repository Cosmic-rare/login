# Discordを使ったログインアプリ

```bash
$ pip install -r requirements.txt
$ npm install
```

```.env
# これはsampleです
TOKEN=la;kjdsl;fkaj;dslkfja;ldksf;aldkfj;alkdsjf;lakjsd;f
```

```bash
$ node index.js
$ python app.py
```

1. Discordで`!button`
2. 出てきたボタンを押す
3. リンクをぽちっと
4. (もう一回そこを開いてもエラーが出ます)
5. `/all`を開くとユーザー名画表示されます
6. (ログインしてないとそう出てきます)
7. リンクをDMで送るようにすればセキュリティーも万全です。
